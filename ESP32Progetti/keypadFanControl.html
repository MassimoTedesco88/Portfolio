Keypad and Display Project for Fan Control

Main goal:Using a keypad, the user can select both the operating time (in seconds) and the fan speed.

By default:

  • Key A corresponds to 15 seconds
  • Key B corresponds to 50% speed
The user can press A to set the fanâ€™s running time, then press B to choose the desired speed level.
To start the program, press #, and to cancel, press *.
Pressing 1 activates one motor, while pressing 2 activates both motors.
All commands and information are displayed on the OLED screen.

Why I built it

I was inspired by the classic keypads often found on industrial machines, and I wanted to see if I could create something similar myself.
