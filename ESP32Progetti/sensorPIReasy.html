Motion Detector with PIR Sensor

A compact motion detection system using a PIR sensor that triggers both sounds and visual alerts. When movement is detected, the buzzer emits a warning tone while the RGB LED flashes alternating read and blue lights, simulating an alarm effect.

Why I built it

I built this small device because I found it both useful and inexpensive to create. It was fun to assemble all the components inside a compact box.
