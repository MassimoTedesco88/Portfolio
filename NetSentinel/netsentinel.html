Suspicious Port & Malware Process Scanner

NetSentinel is a Python-based security tool that monitors active network connections on a Linux system and detects potentially malicious processes based on their behavior and knows malware fingerprints. Given a list of suspicious ports, it scans all current TCP/UDP connections using the ss command, identifies associated processes, and analyzes their binary executable. It computes the SHA256 hash of each process and checks it against a known database of malicious hashes.

Note: The code in this project was written with the help of ChatGPT.

Technologies used

Python, psutil, subprocess, ss

Why I built it

I wanted to explore network security and low-level system monitoring by creating a tool that helps detect suspicious or malicious processes through automated scanning and hash-based verification. It was also an opportunity to improve my skills in system-level scripting and threat detection.

What it does

  • Scans system connections via ss -tunap
  • Maps connections to associated process IDs and executables
  • Computes SHA256 hash of each binary and compares it with a known malware hash list
  • Displays process owner, command path, and hash status

What I learned

This project helped me understand how to interact with the operating system to extract low-level network and process data. I gained experience with secure hashing techniques, process-to-connection mapping, and integrating threat intelligence sources into security tools.
