ESP32 Web Server for LED and Motor Control

A web-based control interface built with ESP32 that allows users to turn LEDs on/off, change RGB colors and control two DC motors through MOSFET.

Why I built it

To explore IoT and learn how to control hardware from a web interface using the ESP32.
