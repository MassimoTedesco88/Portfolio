Python Tool to detect Suspicious Processes

Personal Project | 01.06.2025

Python

A Python-based system diagnostic and security tool that scans all active processes on a local machine, computes the SHA-256 hash of each executable, and checks it against a live malware hash feed from Abuse.ch.

The program extracts system hardware information (CPU,RAM), logs top resource-consuming processes, and saves all results to CSV and log files in the user's Documents folder.

It implements threat comparison logic with a local hash database, automatically updated monthly, ensuring fast and offline-ready performance. Built using psutil and hashlib, the tool demonstrates secure file handling, process introspection, and real-world integration of open threat intelligence - without requiring any user registration or third-party API keys.

Note: The code in this project was written with the help of ChatGPT.

Python, Financial Modeling Prep API, HTML, CSV

I developed a Python-based stock analysus program that leverages the Financial Modeling Prep API to analyze up to 250 tickers per day (free API tier).

The tool performs the following key functions:

  • Data Acquisition: Automatically downloads historical prices and financial data for selected tickers.
  • Analysis & Indicators:
    • Calculates technical, fundamental, and sentiment indicators.
    • Benchmarks each asset against global market conditions.
    • Detects the current market phase (e.g., accumulation, distribution).
    • Estimates investment horizon and automatically sets entry price, take profit, and stop loss levels.
    • Assesses risk and investment attractiveness.
    • Provides commentary in Italian on detected signals and suggests buying opportunities for short, medium, or long-term strategies.
  • Reporting & Strategy:
    • Generates interactive HTML reports and CSV files for each analysis.
    • Clickable links open detailed reports for each analyzed ticker.
    • Ranks and lists the top investment candidates using two custom strategies based on defined indicators and performance thresholds.

This project was developed using Python with the support of AI tools such as ChatGPT to assist in coding. I focused on designing the system, defining its logic, and integrating the various components while learning through the development process.
